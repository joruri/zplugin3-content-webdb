# Zplugin::Content::Webdb

JoruriCMS2017で利用できるデータベースコンテンツプラグインです。
データベースコンテンツの構築に必要な機能をインストールします。

## 想定環境

* OS: CentOS 7.2
* Webサーバ: nginx 1.12
* Appサーバ: unicorn 5.4
* Database: PostgreSQL 9.5
* Ruby: 2.6
* Rails: 5.2
* JoruriCMS 2017 Release4

## インストール

doc/INSTALL.mdを参照してください。
